# Baxter Mortgage — Brand System

Derek Smith (@dereklends) personal brand content pipeline.

## Quick Start

Generate a post:
```
node brand-system/generate-post.js --type jab --index 0
node brand-system/generate-post.js --type education --hook "3 things to do before you apply." --sub "Save this."
node brand-system/generate-post.js --type local --hook "Kennebunk is moving fast."
node brand-system/generate-post.js --type personal
node brand-system/generate-post.js --type urgency --hook "Bought in 2022-2024?"
```

## Post Types

| Type | Day | Purpose |
|------|-----|---------|
| `personal` | Monday | Face, story, trust |

| `local` | Wednesday | York County market |
| `jab` | Thursday | Bank vs broker contrast |
